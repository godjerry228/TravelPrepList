# PWA 圖示說明

## 所需圖示檔案

請準備以下尺寸的圖示檔案：

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## 快速產生圖示

### 方式一：線上工具

1. 訪問 [PWA Builder](https://www.pwabuilder.com/imageGenerator)
2. 上傳一張 512x512 的原始圖片
3. 下載產生的圖示包
4. 解壓縮並將檔案放入此目錄

### 方式二：使用設計工具

推薦圖示設計：
- 主題：旅行箱/清單/勾選圖示
- 背景色：#3b82f6（藍色）
- 圖示色：白色
- 風格：扁平化、簡約

### 方式三：暫時替代（僅測試用）

如果只是要快速測試，可以使用線上產生器：
- [Favicon.io](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)

或使用純色圖片暫代，Windows 使用小畫家建立即可。

## 圖示設計建議

```
┌──────────────┐
│              │
│   ✓  📋     │  <- 旅遊清單主題
│              │
│  Travel      │  <- 可選：文字標示
│              │
└──────────────┘
```

顏色建議：
- 主色：#3b82f6（藍色）
- 次色：#ffffff（白色）
- 強調色：#10b981（綠色，代表完成）

## 檢查清單

產生圖示後，請確認：
- [ ] 所有尺寸圖示都已產生
- [ ] 檔案命名正確
- [ ] 圖示清晰可辨
- [ ] 在不同裝置上測試安裝效果
